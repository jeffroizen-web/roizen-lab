## ACTION ITEMS FOR JEFF

1. **Q1 needs a custom figure.** The causation/MR concept has no existing visual in any PPT. Consider creating a simple arrow diagram: "Gene variant -> Vitamin D level -> Disease outcome" to illustrate the Mendelian Randomization logic.

2. **Q3 is blocked by WMF format.** The CYP2R1 KO data figures are all Windows Metafiles. Options:
   - Re-export the charts from the original Excel/Prism files as PNG
   - Use a WMF-to-PNG converter (LibreOffice can do this)
   - Screenshot the slides directly from PowerPoint

3. **Dark backgrounds.** Most data figures (2026 talk, Lilly talk) have black backgrounds. For the purple-gold website, consider:
   - Using as-is with appropriate container styling (dark card on light page)
   - CSS `mix-blend-mode` or `filter: invert()` tricks
   - Re-exporting from source with white backgrounds

4. **Resolution check.** Some figures from the 2026 talk are modest resolution (509x314 for lean mass, 511x316 for fat mass). They will look acceptable at web sizes but not crisp on retina displays. The Lilly talk figures are much higher resolution (2000-3000px wide).

5. **The mouse silhouette** (`paradigm-figures_slide11_img1.png`) is a cute black mouse icon — not a data figure, but could be useful as a decorative element or icon on the site.
